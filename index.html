HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The meta element contains specific metadata information such as typeface, compatibility, etc.
  • ul: unordered list
  • li: bullets
  • headers only go up to h6
  • here is a great resource for HTML markups.
  • "!DOCTYPE html" tells the browser what document type it should expect when opened.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • * applies to all elements on a page.
  • The display property is assigned with the block value, which assigns each image to start on a new line.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git checkout without additional modifiers can move to the specified branch.
  • git add -A: commits/stages all files in the working branch. (add = saves? commit = uploads?)
  • git commit -m "text": commits the changes to git.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • The const keyword allows us to create variables that can't be reassigned a value. It essentially "locks down" what that variable will be, and once we declare it, we can never reassign it.
  • "script src="./assets/script.js" to link a .js to index at footer
  • Control flow is the order in which a script is executed.
  • falsy = the following values: "0n, 0, -0, null, undefined, NaN"
  • (==) = does not equal, (===) does equal
  • to access a specific item of an Array, the relevant index number must be placed in the following format: name[0] with "name" being the name of the Array and "0" being the index number, beginning from 0.
  • We can find the length of an array by using a built-in property of an array, appropriately named length.
  • The code x++ is a shortcut way of writing x+1. Now every time the code loops, x will increase by 1.
  • when submitting a console.log at the end of a for loop, remember to add the index[x] to log one singular unit at a time.
